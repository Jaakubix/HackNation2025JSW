# 7) Pakowanie i aplikacja web (Python)
- Skrypty CLI:
  - `src/train.py`: argumenty ścieżek i hparamów; zapis checkpointów do `models/`.
  - `src/infer_video.py`: wejście wideo/folder, model checkpoint, zapis CSV/alertów/wizualizacji.
- Pakowanie: `setup.cfg`/`pyproject` (opcjonalnie) lub prosty `requirements.txt`; ujednolić ścieżki w `config.py`.
- Aplikacja web (Python): FastAPI + prosty frontend (HTML/JS/CSS) serwowany przez FastAPI lub alternatywnie Streamlit.
  - Endpointy: upload wideo, uruchom inferencję, zwróć CSV/alerty, podgląd wizualizacji.
  - Widok klienta: lista nagrań, status analizy, tabela cykli z alertami, link do podglądu klatek z bboxami.
  - Uruchamianie: `uvicorn src.web.app:app --reload` (dev) lub `streamlit run src/web/app.py` (jeśli wybierzemy Streamlit).
- Dokumentacja: sekcja README z komendami (conda create/activate, pip install pakietów w env, train, infer, uruchomienie web), przykładowe użycie z folderem testowych nagrań.
- Logging: rotujące logi do `outputs/logs/`; konfiguracja w `logging.yaml` (opcjonalnie).
